# LAMP E-commerce Store

A basic e-commerce web application built using the LAMP stack (Linux, Apache, MySQL, PHP) with GitHub Actions CI/CD.

## Features (Planned)
- Product listing
- Shopping cart
- Checkout flow
- Admin product management
- Unit & integration tests
- CI/CD with GitHub Actions

## Tech Stack
- PHP 8
- MySQL
- Apache
- PHPUnit
- Composer
